你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。

三数之和

给定一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a

  • b + c = 0 ？找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组。

通过上面的例子我们可以看到数组中是有重复的值的，因此我们需要多一步去重的步骤。

其实很好理解，去重的方法就是从小到大排序，然后遍历，遇到重复的就跳过去就行了，需要注意的是每一层都需要去重。

内层循环，如果 sum > target 说明大了，需要 r 左移，如果 sum < target 说明小了，需要 l 右移（因为整个数组是从小到大排列的，左移变小，右移变大）
